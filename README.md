## Overview

Comprehensive documentation for the ZeroKnow IPSwap protocol built on **Stellar's Soroban** smart contract platform.

**Stellar Integration Features:**
- 🌟 **Soroban Smart Contracts** — Three interconnected contracts (atomic_swap, ip_registry, zk_verifier)
- 💰 **USDC Payments** — Native Stellar Asset Contract integration
- 🔒 **Atomic Transactions** — Guaranteed swap execution or full rollback
- 📊 **Persistent Storage** — Stellar storage with TTL management
- 🔐 **ZK Proof Verification** — On-chain Merkle proof validation using Poseidon hash
